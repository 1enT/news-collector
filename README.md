# news-collector
News Collector – это сервис, который помогает выкладывать посты в Telegram канал его владельцу. 
Сервис состоит из 3 частей.
1. Новостного агрегатора, собирающего последние новости с некоторых сайтов СМИ, используя ключевые слова.
2. Интерфейса пользователя, позволяющего отсеивать ненужные новости, с возможностью редактирования отдельных новостей
3. Бота, который непосредственно постит выбранные пользователем новости в Telegram канал.

News Collector будет полезен администраторам Telegram каналов организаций. Поскольку в таких Telegram каналах зачастую содержатся не только внутренние новости организации, но и новости из вне (такими каналами, например, являются каналы гос. учреждений). Данный сервис избавит от необходимости проходить каждый новостной ресурс в поиске нужной новости и будет агрегировать новости с заранее отобранных сайтов и удовлетворяющих некоторым критериям (ключевые слова).

Представим, что Вы – администратор Telegram канала некоего бренда, представленного в вашем регионе.
Вы тратите много времени на Ваш Telegram канал.
Зачем? Потому что нужно выделять время на то, чтобы искать новости о Вашем бренде на сайтах СМИ.
